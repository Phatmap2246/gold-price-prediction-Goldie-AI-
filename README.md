# Gold Price Prediction with LSTM and Sentiment Analysis

Dự án dự đoán giá vàng sử dụng LSTM kết hợp phân tích cảm xúc từ tin tức FED.

## Cài đặt
- Python 3.8+
- Các thư viện: `pip install -r requirements.txt`

## Sử dụng
Chạy notebook `Goldie_AI.ipynb` trên Google Colab hoặc chạy script `goldie_pipeline.py`.

## Dữ liệu
- Giá vàng: yfinance (GLD)
- Tin tức: RSS FED
- Chỉ số vĩ mô: FRED API (tuỳ chọn)

## Kết quả
Biểu đồ và dự đoán được lưu vào Google Drive và Google Sheet.
